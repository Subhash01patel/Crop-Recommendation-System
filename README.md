# Crop-Recommendation-System

A simple Machine Learning project that recommends the best crop to grow based on soil and weather conditions.

---

## About the Project

Choosing the right crop is important for better yield. This system takes basic soil nutrients and climate values as input and predicts the most suitable crop using a trained machine learning model.

---

## Features

* Recommends suitable crop
* Uses soil and weather parameters
* Easy to understand and use
* Useful for academic and learning purposes

---

## Input Parameters

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH value
* Rainfall

---

## Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn

---

## How to Run

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/Crop-Recommendation-System.git
   ```

2. Install required libraries

   ```bash
   pip install -r requirements.txt
   ```

3. Run the project

   ```bash
   python app.py
   ```

---

## Output

The system displays the recommended crop based on the given input values.
